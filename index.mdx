# From zero to one

Dev0 is an agentic engineering platform that turns your product ideas into reviewable GitHub pull requests. You describe what you want to build — Dev0 plans the work, executes tasks inside isolated sandboxes using Gemini AI, and ships each task as a PR ready for your review.

Engineers stay in control as technical leads. The platform handles execution.

## What Dev0 does

<Steps>
  <Step title="Describe your idea">
    Type a rough description of the feature or product you want to build. Dev0's **Preview Agent** (Gemini Flash) converts it into a structured project name, summary, and tech stack in seconds.
  </Step>
  <Step title="Generate a task plan">
    Dev0's **Planner Agent** (Gemini Pro) produces a dependency-aware task graph — breaking the project into ordered phases, each task assigned an appropriate model and execution strategy.
  </Step>
  <Step title="Execute in isolation">
    Each task runs inside a dedicated **E2B sandbox** with Gemini CLI operating in YOLO mode. The agent writes code, runs commands, and commits changes — all in a contained, reproducible environment.
  </Step>
  <Step title="Stream live feedback">
    JSONL events from the sandbox are parsed and streamed to the **Mission Control UI** in real time via Upstash Realtime. You see every tool call, file write, and terminal output as it happens.
  </Step>
  <Step title="Review and merge">
    Every completed task opens a **GitHub pull request** on your project repository. Approve, request changes, or merge directly from the Dev0 review interface — no context switching required.
  </Step>
</Steps>

## Key differentiators

<CardGroup cols={2}>
  <Card title="Platform-controlled autonomy" icon="shield-check">
    Agents run inside E2B sandboxes with strict orchestration and guardrails. No runaway processes.
  </Card>
  <Card title="PR-first workflow" icon="code-pull-request">
    Every task ends with a reviewable GitHub pull request — humans stay in control of what merges.
  </Card>
  <Card title="Real-time visibility" icon="wave-pulse">
    Live task logs and status updates stream directly to the UI via Upstash Realtime pub/sub.
  </Card>
  <Card title="Gemini-powered execution" icon="sparkles">
    Gemini 3 Pro handles planning and Gemini 3 Flash handles fast execution — the right model for each job.
  </Card>
  <Card title="Human-in-the-loop" icon="user-check">
    Approve, request changes, or merge PRs from the Mission Control UI without leaving the platform.
  </Card>
  <Card title="Security by design" icon="lock">
    Secrets are injected via environment variables and automatically redacted from all execution logs.
  </Card>
</CardGroup>
